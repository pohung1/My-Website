Titanic Survival Prediction

  • Click Here To View Project
    I analyzed the Titanic dataset from Kaggle to predict survival. My results suggest survival was higher for women, children, and first-class passengers—especially those with higher fares and smaller family sizes. Adult third-class men traveling alone were least likely to survive.

Spam/Ham Email Classification

  • Click Here To View Project
    I built a machine learning model to classify emails as span or non-spam(ham) using Pandas, Python, and scikit-learn. I performed text preprocessing and feature engineering on a dataset of over 8,000 labeled emails, then trained logistic regression classifier and evaluated model performance using cross-validation. The project demonstrates how text features can be used to detect spam patterns in email content.
